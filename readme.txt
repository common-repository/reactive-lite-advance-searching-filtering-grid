Reactive LITE is a search filtering and grid plugin for your wp site.

== Description ==
Reactive LITE gives you option to create any kind of post search, filtering and grid. It has very easy drag&drop searching, filtering and grid block builder via shortcode.

Add `[reactive key="123"]` this shortcode into your page and you can then add your builder from the front-end. The key must be an unique integer value.

= The features are offered by this plugin =

*   Add grid block
*   Add search block
*   Checkbox search
*   Radiobox search
*   Textbox search
*   2 grid layout
*   Drag & Drop Functionality for page building
*   Multilingual ( 6 translation included )

== Installation ==

= Minimum Requirements =

* WordPress 4.4 or greater
* PHP version 5.4.0 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Reactive LITE, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type Reactive LITE and click Search Plugins. Once you’ve found our search & filter pro plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our eCommerce plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Frequently Asked Questions ==

= Where to add shortcode for reactive builder? =

Log in to your WordPress dashboard, navigate to the Pages menu and click Add New.

= Which shortcode to add for reactive builder? =

`[reactive key="123"]`. Make sure this key is unique integer.

= How to create the search page? =

`[reactive key="123"]` after adding this shortcode into your page. Click view page and in the front-end your add add your search and grid block.
